Agenda에 따른 내용
1. 서울대학교 장병탁 교수
생성형 AI로 촉발된, 모든 영역에 AI가 적용되는 AI 전환의 시대를 맞아 국내,외 기업의 AI 전환 트렌드를 설명하고 기업에 성장에 있어 준비해야 될 부분들을 설명함
AI는 결국 지각, 사고, 행동 과거에는 data를 넣었다면 현재는 '학습'의 원리로 시행착오를 통해 딥러닝 시키는 것이 중요함
결론적으로 Embodied Intelligence / 데이터를 모으고 그 데이터가 실제 데이터 기반임이 중요함

2. BI Matrix 배영근 대표이사
NO Code 플랫폼을 강조하며 현재 low code 플랫폼 까지 오는 과정에 대한 설명 g-matrix와 같은 경우 수천 개의 데이터베이스를 의미 있는 정보를 차장서 분석하고 표현하는데에 중점을 둠
결국 low code는 정형적인 UX업무에 사용되고, no code는 비정형적인 UX업무에 사용된다면 결국 비정형 업무를 AI를 사용해 개발 하는 것을 목표로 삼음

3. BI Matrix 전규화 이사 
두개의 디자이너에 대한 설명으로 첨부파일 참조

4,5,6,7 -> BI Matrix 협력 회사로 그에 있어 제품을 사용해보며 후기들에 대한 공유 그리고 현재 시장에 돌아가는 인사이트등을 배우고 느낄 수 있었음.
